# Utility Scripts

This directory contains utility scripts for data analysis and debugging.

## Scripts

### `check_stats.py`
Analyzes cleaned data statistics.

**Usage:**
```bash
python scripts/check_stats.py
```

**Output:**
- Page type distribution
- Content length statistics
- Sample cleaned content

### `list_gemini_models.py`
Lists available Gemini models for content generation.

**Usage:**
```bash
python scripts/list_gemini_models.py
```

**Output:**
- Model names
- Supported generation methods

## Notes

These scripts are for development and debugging only. They are not required for production deployment.
